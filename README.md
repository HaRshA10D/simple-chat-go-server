## Description
A simple group chat application server written in GoLang.

## Dev environment setup
* Install GoLang from [here]()
* GoDep for dependencies
```
$ brew install dep 
```
* `make db_create`
* `make db_migrate`
* `make build`

## Run instructions
To run the application 
```
$ ./out/chat-server
```

## Run tests
To run all test `make test`
